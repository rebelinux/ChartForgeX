# Network Security Icons

Source: original hand-authored ChartForgeX topology SVG artwork.
Workflow: edit `svg/*.svg` as the source of truth, then refresh `previews/*.png` and validation reports with `ChartForgeX.Tools.IconImport --refresh-pack`.
License: ChartForgeX repository license, see `LICENSE` in this folder.

This pack keeps reviewable `manifest.json`, `svg/*.svg`, and generated `previews/*.png` side-by-side. It intentionally uses generic network security role names. Product-specific or vendor-branded packs should live in separate auditable folders with explicit provenance and license notes.
